# Quick Character Reference

Pull up a concise reference card for a character without disrupting the scene.

## Find the Character

The user provided: **$ARGUMENTS**

- If a name was given, search `scene-partner/characters/` for a matching file (case-insensitive, partial match is fine).
- If no name was given and there are only 2-3 characters, list them and ask which one.
- If no name was given and there are many characters, list all available characters and ask.
- If the character doesn't have a file, say so and offer to create one with `/character`.

## Read and Summarize

Read the character's file and present a compact reference card — **not** the full file. Extract and display:

### [Character Name]
- **Appearance:** 2-3 key visual details
- **Personality in brief:** 3-5 words capturing their core energy
- **Voice/Speech:** How they talk, any verbal tics or patterns
- **Key dynamics:** How they relate to other characters in the current scene
- **Abilities:** Anything relevant to the current scenario (skip if not applicable)
- **Soft spots / Triggers:** What gets under their skin, what disarms them

## Rules

- Keep it short. This is a mid-scene glance, not a deep dive.
- Use bullet points, not paragraphs.
- Don't include full backstory unless specifically asked.
- Don't break the RP mood — present it cleanly as a reference sidebar, then let the user get back to the scene.
